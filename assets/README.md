# Audio Assets

## completion.mp3

Place your completion sound file here. The bot will play this sound when a Deep Work session completes.

### Requirements:
- Format: MP3 (recommended) or other formats supported by FFmpeg
- Duration: 3-10 seconds recommended
- Volume: Moderate (not too loud)

### Where to Find Completion Sounds:

**Free Sound Resources:**
- [Freesound.org](https://freesound.org) - Search for "bell", "chime", or "notification"
- [Zapsplat](https://www.zapsplat.com) - Free sound effects
- [Mixkit](https://mixkit.co/free-sound-effects/) - Free sound effects

**Suggested Search Terms:**
- "success bell"
- "achievement chime"
- "notification tone"
- "pleasant alert"
- "meditation bell"

### Creating Your Own:

You can create a simple completion sound using:
- GarageBand (Mac)
- Audacity (Free, cross-platform)
- Online tone generators

### Note:

If no `completion.mp3` file is present, the bot will log an error but continue to function normally (just without audio).
